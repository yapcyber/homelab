# Yanis Deschamps

*Cybersecurity apprentice — detection, infrastructure, and security operations*

Master's student in Cybersecurity, working as a cybersecurity administrator on a
work-study contract. I operate at the intersection of detection (SOC, SIEM,
detection-as-code), infrastructure (virtualization, network segmentation,
hardening), and day-to-day security operations. My approach: building systems
that are secure by default, where a failure fails in a controlled way rather than
silently.

This site documents my detection homelab and my technical write-ups — how I
design, troubleshoot, and solve, not just the tools I use.

## Skills

- **Detection / SOC** — SIEM (Wazuh), NIDS (Suricata, Zeek), detection-as-code (Sigma), MITRE ATT&CK mapping
- **Infrastructure & networking** — Proxmox, OPNsense, VLAN segmentation, reverse proxy, VPN, infrastructure-as-code (Packer, OpenTofu, Ansible)
- **Security operations** — incident response (TheHive, Cortex), vulnerability assessment (OpenVAS), hardening, OSINT
- **Automation** — Python, CI/CD, GitOps, systematic fail-closed approach

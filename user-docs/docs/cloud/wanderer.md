# Wanderer

Wanderer est votre carnet de randonnées : traces GPX, itinéraires et sorties, sur
une carte.

## À quoi ça sert

- Enregistrer et consulter vos randonnées et traces GPX.
- Planifier des itinéraires et garder des listes de sorties.

## Comment l'utiliser

1. Activez WireGuard puis ouvrez `https://wanderer.yapserver.fr` et connectez-vous
   avec **Authentik**.
2. Importez une trace GPX ou créez un itinéraire directement sur la carte.

## Interactions

- **Connexion unique (Authentik)**.
- **Complémentaire de Dawarich** : Dawarich pour l'historique automatique de
  position, Wanderer pour les randonnées choisies.
- **WireGuard** requis.

## Limites

- L'import attend des fichiers de trace standard (GPX) ; d'autres formats peuvent
  demander une conversion préalable.
- Le fond de carte dépend de sources externes : certaines couches peuvent être
  temporairement indisponibles.

# Dépannage utilisateur

1. Vérifier que l'appareil a accès à Internet.
2. Confirmer que WireGuard est actif et que son handshake est récent.
3. Retester dans une fenêtre privée ou un autre navigateur.
4. Vérifier si un seul service ou tous les services sont affectés.
5. Noter l'heure, l'URL, le message exact et l'appareil utilisé.

Ne désactivez pas les contrôles TLS et ne communiquez jamais mot de passe, code
MFA ou clé WireGuard dans une demande d'assistance.

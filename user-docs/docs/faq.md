# Questions fréquentes

## Pourquoi le site fonctionne uniquement avec WireGuard ?

La documentation décrit des services privés. Elle est volontairement limitée
aux réseaux d'administration et au VPN.

## Puis-je partager mon compte ?

Non. Demandez la création d'un compte individuel afin de préserver les profils,
les droits et la traçabilité.

## La page signale un certificat non reconnu

N'ajoutez pas d'exception permanente. Notez le nom du site, l'appareil et l'heure
puis contactez l'administrateur.

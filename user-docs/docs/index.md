# Documentation utilisateur

Cette documentation explique comment utiliser les services YapServer sans
exposer les procédures réservées à l'administration du homelab.

## Pour commencer

1. Connectez WireGuard avant d'ouvrir un service YapServer.
2. Utilisez l'adresse communiquée pour le service concerné.
3. Conservez vos identifiants dans un gestionnaire de mots de passe.
4. En cas d'erreur, consultez d'abord la page du service puis le
   [guide de dépannage](./depannage.md).

Les premiers guides disponibles concernent Jellyfin, Navidrome et Grafana. Les
autres services seront ajoutés progressivement après validation utilisateur.

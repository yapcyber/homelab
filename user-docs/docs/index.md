# Documentation utilisateur

Cette documentation explique comment utiliser les services YapServer sans exposer
les procédures réservées à l'administration du homelab.

## Pour commencer

1. Connectez **WireGuard** avant d'ouvrir un service YapServer.
2. Utilisez votre **compte unique** (connexion Authentik) là où il est proposé.
3. Conservez vos identifiants dans Vaultwarden, le gestionnaire de mots de passe.
4. En cas d'erreur, consultez d'abord la page du service, puis le
   [guide de dépannage](./depannage.md).

## Comment naviguer

Les guides sont classés par usage :

- **Accès & compte** : se connecter au VPN et au compte unique — à lire en premier.
- **Médias** : films, séries, musique, livres audio et lecture.
- **Cloud & vie perso** : fichiers, photos, mots de passe et services du quotidien.
- **Finances** : budget et suivi de patrimoine.
- **Portail & supervision** : point d'entrée, état des services et notifications.
- **Cloud gaming** : jouer à distance via le streaming.

Chaque page décrit ce que fait le service, comment l'utiliser, ses interactions
avec les autres services et ses limites.

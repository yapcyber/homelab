# Kavita

Kavita est la liseuse en ligne pour bandes dessinées, mangas, romans et documents
de la médiathèque.

## À quoi ça sert

- Lire BD, mangas, comics et ebooks dans le navigateur.
- Suivre votre progression et constituer des listes de lecture.

## Comment l'utiliser

1. Activez WireGuard puis ouvrez `https://kavita.yapserver.fr`.
2. Connectez-vous avec **Authentik** (compte unique).
3. Parcourez les bibliothèques et ouvrez un titre : la lecture reprend où vous
   vous étiez arrêté.

## Interactions

- **Connexion unique (Authentik)**.
- **Complémentaire de Readeck** : Readeck pour les articles web, Kavita pour les
  livres et BD.
- **WireGuard** requis.

## Limites

- La lecture se fait principalement dans le navigateur ; les applications OPDS
  tierces sont optionnelles.
- Un ouvrage manquant ou mal classé se signale avec le titre attendu.

# Navidrome

Navidrome donne accès à la bibliothèque musicale depuis un navigateur ou une
application compatible Subsonic.

## Connexion

1. Activez WireGuard.
2. Ouvrez `https://navidrome.yapserver.fr` ou configurez cette URL dans le client.
3. Utilisez votre compte personnel.

Les playlists et favoris appartiennent au compte. Une piste absente ou mal
identifiée doit être signalée avec l'artiste, l'album et le titre attendus.
